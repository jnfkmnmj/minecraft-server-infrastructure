# Backup Scripts

This folder contains scripts for creating, organizing, validating, and maintaining Minecraft server backups.

## Contents

- **backup-world.sh** – Creates backups of the Minecraft world.
- **backup-config-files.sh** – Backs up server configuration files.
- **create-backup-directory.sh** – Creates the required backup directory structure.
- **archive-logs.sh** – Archives server log files.
- **clean-old-backups.sh** – Removes outdated backup files.
- **check-backup-files.sh** – Verifies the integrity and availability of backup files.

These scripts are intended as reference automation utilities and may require environment-specific adjustments before deployment.
