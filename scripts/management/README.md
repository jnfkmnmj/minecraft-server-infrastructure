# Management Scripts

This folder contains scripts for starting, stopping, updating, and maintaining the Minecraft Forge server.

## Contents

- **start-server.sh** – Starts the Minecraft server.
- **stop-server.sh** – Stops the Minecraft server safely.
- **restart-server.sh** – Restarts the server.
- **auto-restart-if-crashed.sh** – Automatically restarts the server after unexpected crashes.
- **update-server-files.sh** – Updates server files and related components.
- **verify-java-installation.sh** – Verifies that the required Java environment is installed correctly.

These scripts are intended as reference automation utilities and may require environment-specific adjustments before deployment.
