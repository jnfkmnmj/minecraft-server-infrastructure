# Monitoring Scripts

This folder contains scripts for monitoring server health, generating reports, and performing diagnostic checks.

## Contents

- **monitor-server.sh** – Monitors overall server status.
- **system-summary.sh** – Displays a summary of system resources and server information.
- **generate-server-report.sh** – Generates a server status report.
- **list-server-processes.sh** – Lists running server-related processes.
- **check-server-status.sh** – Checks whether the server is running.
- **check-server-uptime.sh** – Displays server uptime information.
- **check-server-files.sh** – Verifies required server files.
- **check-log-files.sh** – Checks server log files.
- **check-crash-reports.sh** – Checks for crash reports.
- **check-network-connections.sh** – Displays active network connections.
- **check-disk-space.sh** – Checks available disk space.

These scripts are intended as reference automation utilities and may require environment-specific adjustments before deployment.
