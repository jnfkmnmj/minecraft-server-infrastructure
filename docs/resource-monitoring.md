# Resource Monitoring

## Overview

This document records methods used to monitor server resource usage on Ubuntu Linux.

## Common Commands

### CPU usage

```bash
top
```

### Memory usage

```bash
free -h
```

### Disk usage

```bash
df -h
```

### Process monitoring

```bash
htop
```

## Common Issues

- High RAM usage
- CPU spikes
- Disk space limitations
- Increased server lag during heavy mod usage

## Goal

Monitor server resources to improve stability and multiplayer performance.
