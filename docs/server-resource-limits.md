# Server Resource Limits

## Overview

This document records common resource limitations encountered while managing Minecraft Forge servers.

## Common Limitations

- Limited RAM
- CPU bottlenecks
- Disk space limitations
- Network latency
- Slow world generation

## Example Symptoms

- Server lag
- Delayed chunk loading
- Player disconnects
- Long startup times
- Increased crash frequency

## Monitoring Commands

### Memory usage

```bash
free -h
```

### Disk usage

```bash
df -h
```

### CPU usage

```bash
top
```

## Goal

Identify server limitations early and improve infrastructure stability.
