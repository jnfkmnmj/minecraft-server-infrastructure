# Security Notes

## Overview

This document records basic security practices for managing a Minecraft server on Ubuntu Linux.

## Basic Practices

- Keep the system updated
- Allow only required firewall ports
- Use SSH instead of a graphical interface
- Avoid running unnecessary services
- Keep regular backups

## Important Ports

```text
22    SSH
25565 Minecraft Server
```

## Goal

Maintain a safer server environment while allowing players to connect properly.
